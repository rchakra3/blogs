```
usage: zip_upload.py [-h] [-p PATH] [-o OUT] [-n ACCOUNT_NAME]
                     [-c CONTAINER_NAME] [-k ACCOUNT_KEY]

Zip and upload file to blob storage

optional arguments:
  -h, --help            show this help message and exit
  -p PATH, --path PATH  path to directory you want to zip
  -o OUT, --out OUT     name of output file
  -n ACCOUNT_NAME, --account ACCOUNT_NAME
                        name of the storage account
  -c CONTAINER_NAME, --container CONTAINER_NAME
                        name of storage container
  -k ACCOUNT_KEY, --key ACCOUNT_KEY
                        storage account key
```
